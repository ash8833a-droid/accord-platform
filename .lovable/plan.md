# تنظيم الفرق بين "بوابتي" و"اللجان" و"ردود اللجان"

## التشخيص الحالي

| الصفحة | الدور الحقيقي | المشكلة |
|---|---|---|
| `/portal` "بوابتي" | عرض شخصي موحّد (Read-Only) | يبدو مكرراً مع اللجان، لا يميّز نفسه بصرياً |
| `/committee/{type}` "اللجان" | مكان التنفيذ الفعلي (Read-Write) داخل لجنة واحدة | منفصلة تماماً عن بوابتي، يحتاج العضو الانتقال يدوياً |
| `/task-responses` "ردود اللجان" | لوحة رقابية للإدارة والجودة فقط | اسمها يوحي أنها للجميع |

## الفلسفة الجديدة (3 طبقات واضحة)

```text
طبقة 1 — بوابتي  (شاشتي الشخصية)
   ↓ "ابدأ العمل في..."
طبقة 2 — اللجنة  (ورشة العمل التنفيذية)
   ↓ ترفع التقارير لـ
طبقة 3 — الإدارة العليا/ردود اللجان (الرقابة)
```

كل عضو يبدأ يومه من **بوابتي**، يرى ما يخصّه، ثم يدخل بضغطة على بطاقة المهمة مباشرة إلى اللجنة المعنية لتنفيذ الإجراء.

## التعديلات المقترحة

### 1. توضيح هوية كل صفحة بشريط علوي مميّز
- **بوابتي**: شارة "عرض شخصي · للقراءة" + جملة "اضغط أي مهمة للانتقال إلى لجنتها"
- **صفحة اللجنة**: شارة "ورشة العمل التنفيذية" مع اسم اللجنة بارزاً
- **ردود اللجان**: شارة "لوحة الإدارة والجودة" + قصرها على Admin/Quality في القائمة الجانبية

### 2. إضافة Quick Actions داخل بوابتي (بدون مغادرة الصفحة)
حالياً بوابتي عرض فقط. سنضيف:
- **زر "+ مهمة جديدة"** يفتح Dialog لاختيار اللجنة ثم إنشاء المهمة مباشرة
- **زر "+ طلب صرف"** يفتح Dialog مماثل
- زر "تقرير PDF" (موجود)

هذا يحوّل بوابتي من شاشة عرض إلى **مركز تحكّم شخصي حقيقي**، فلا يحتاج العضو لمغادرتها للأعمال البسيطة.

### 3. تنظيف القائمة الجانبية للأعضاء العاديين
الحالة الحالية للعضو المقيّد:
```
بوابتي · بنك الأفكار · التواصل
```
ممتاز كما هو — لا تغيير.

العضو الكامل يرى حالياً صفحات متفرقة (`/payment-requests`, `/reports`) قد تتداخل مع بوابتي. سنجمعها تحت قسم واحد:
```
بوابتي  ← الافتراضية
لجاني  ← قائمة منسدلة بأسماء لجان العضو فقط
─────
بنك الأفكار · سجل العرسان · التواصل
─────
[للإدارة فقط] ردود اللجان · التقارير والجودة · فريق العمل · الإدارة العليا
```

### 4. إضافة "Empty State" تعليمي في كل صفحة
- بوابتي للعضو الجديد بدون لجان: شرح "هذه شاشتك الشخصية، ستظهر هنا مهامك حال انضمامك للجنة"
- صفحة اللجنة لمن لا ينتمي إليها: زر "اطلب الانضمام"

### 5. إعادة تسمية بسيطة للوضوح
- `ردود اللجان` → **`متابعة أداء اللجان (إدارة)`** — يوضح أنها للرقابة
- لا تغيير على بوابتي/اللجان

## ما يحصل عليه العضو في النهاية

| المهمة | من أين أعملها؟ |
|---|---|
| أرى مهامي اليوم | **بوابتي** |
| أنشئ مهمة سريعة | **بوابتي** (Dialog) أو داخل اللجنة |
| أحرّك مهمة بين أعمدة Kanban | **داخل اللجنة** (السحب والإفلات) |
| أرفع طلب صرف | **بوابتي** (Dialog) أو داخل اللجنة |
| أعتمد طلب صرف | **داخل اللجنة** (للرئيس فقط) |
| أرفع تقرير شامل | **داخل اللجنة** |
| أصدّر تقرير PDF شخصي | **بوابتي** |
| أراجع أداء كل اللجان | **متابعة أداء اللجان** (إدارة فقط) |
